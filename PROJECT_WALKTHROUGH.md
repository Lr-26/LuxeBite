# 🍷 LuxeBite Restaurant: Proyecto Completo

He desarrollado un sitio web profesional y completo para **LuxeBite**, una experiencia gastronómica de alta gama. El proyecto cumple con todos los requisitos de diseño premium, animaciones fluidas y funcionalidad full-stack.

## 🚀 Arquitectura Técnica

### Frontend (`/frontend`)
- **Framework**: Vite + React + TypeScript.
- **Animaciones**: **Framer Motion** para revelaciones al hacer scroll y efectos de entrada suaves.
- **Diseño**: Sistema visual basado en **Glassmorphism**, con una paleta de color Carbón Profundo y Oro (#D4AF37).
- **Tipografía**: `Playfair Display` para títulos elegantes y `Outfit` para legibilidad moderna.
- **Iconos**: Lucide React.

### Backend (`/backend`)
- **Framework**: Node.js + Express + TypeScript.
- **Endpoints**:
  - `GET /api/menu`: Entrega los platos categorizados (Entradas, Platos Fuertes, Postres).
  - `POST /api/reserve`: Gestiona las solicitudes de reserva de mesa.
- **Servidor**: Configurado por defecto en el puerto `5001`.

## ✨ Características Principales
1. **Hero de Impacto**: Fondo cinematográfico con superposición de gradiente y botones de llamada a la acción.
2. **Menú Dinámico**: Los platos se cargan desde el backend y se presentan en tarjetas de "cristal" con efectos de elevación al pasar el mouse.
3. **Formulario de Reservas**: Integrado con validación y estados de envío.
4. **Respuesta Adaptativa**: Diseño optimizado para dispositivos móviles y escritorio.

---

## 🛠️ Cómo ejecutar el proyecto

### 1. Iniciar el Backend
```bash
cd backend
npm run dev
```

### 2. Iniciar el Frontend
```bash
cd frontend
npm run dev
```

> [!TIP]
> También puedes usar el comando `npm run dev` desde la raíz para iniciar ambos simultáneamente si los procesos están configurados.
